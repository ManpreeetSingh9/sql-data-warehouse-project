# 📊 Data Warehouse and Analytics Project

This project demonstrates an end-to-end **data warehousing and analytics solution**, including data ingestion, transformation, modeling, and reporting. It is designed to showcase practical skills in **data engineering and analytics**.

---

## 🏗️ Data Architecture

The project follows the **Medallion Architecture**:

### 🟤 Bronze Layer
- Stores raw data from source systems (ERP and CRM)
- Data is ingested in its original format

### ⚪ Silver Layer
- Cleans and transforms raw data
- Handles missing values, duplicates, and inconsistencies
- Standardizes data for further processing

### 🟡 Gold Layer
- Contains business-ready data
- Structured using a **star schema**
- Optimized for analytics and reporting

## 📖 Project Overview

This project includes:

- **Data Ingestion**
  - Importing CSV data into the data warehouse

- **ETL Pipelines**
  - Extract, transform, and load processes across layers

- **Data Modeling**
  - Designing fact and dimension tables

- **Analytics & Reporting**
  - Generating SQL-based insights for decision-making
